Криптографические средства защиты информации

Тема

Методы криптографической защиты информации

Ключевые слова

Стенография
наука о сокрытия факта наличия или передачи информации

Тайнопись
способ шифрования данных, который дает возможность замаскировать присутствие самих данных

Микроточки
использование сверхминиатюрной фотографии для сокрытия информации

Шифрование (расшифрование)
процесс преобразования информации в секретный код, который имеет нечитаемый вид

Секретный ключ шифрования
специальный набор данных, при помощи которых криптографическим алгоритмом выполняется шифрование и расшифровка информации

Криптография
наука, изучающая методы и средства защиты информации

Шифр Цезаря
самый простой метод, при котором каждая буква сообщения заменяется на другую, находящейся на определенном количестве позиций правее или левее в алфавите

Шифр Виженера
более сложный метод, который использует ключевое слово для генерации последовательности ключей, которые затем используются для шифрования каждого символа сообщения

Хэш-функция
математический алгоритм, преобразующий тексты различной длинны в одну строку, состоящую из цифр и букв фиксированной длинны

Аутентификация
определение объекта, по известной только ему информации

Рекомендации для обучающегося (самостоятельная работа)

  • Просмотрите видеолекцию по теме «Методы криптографической защиты информации».
  • Если в процессе изучения материала возникнут вопросы, запишите их для дальнейшего обсуждения с преподавателем и/или с группой обучающихся.
  • При необходимости прослушайте/просмотрите материал еще раз.
    Возвращайтесь к наиболее сложным аспектам темы.
  • Соблюдайте здоровьесберегающий режим: чередуйте работу с электронными носителями с отдыхом и гимнастикой для глаз.

Рекомендации для обучающегося (самостоятельная работа)

  • Запустите интерактивный тренажер по теме «Криптографические методы защиты информации».
  • В ходе практической работы выполните тренировочные задания, которые помогут освоить новый материал.
  • Уделите особое внимание правильной организации рабочего места.
  • Выполните тренировочные задания на проверку первичного освоения материала.
  • При выполнении заданий обратите внимание на рекомендации преподавателя.
  • При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.

Рекомендации для обучающегося (самостоятельная работа)

  • Запустите интерактивный тренажер по теме «Криптографические способы защиты информации
  • Последовательно ознакомьтесь с условиями учебной задачи.
    Выполните задания интерактивного тренажера. Обратите внимание на отсутствие подсказок и ограниченное число попыток.
  • Выполните тренировочные задания на проверку приобретенных знаний, умений и навыков:
    - интерактивное задание с выбором одного правильного ответа;
    - интерактивное задание с выбором нескольких правильных ответов;
    - интерактивное задание на установление соответствия;
    - интерактивное задание на упорядочивание.
  • Осуществите самопроверку и предварительную оценку.
    Используйте технологию пошагового контроля по результату за каждое задание.
    Сформулируйте конечный результат своей работы.
  • При возникновении ошибок возвращайтесь к повторному изучению теоретического материала.
