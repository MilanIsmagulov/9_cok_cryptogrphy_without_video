При выборе программного обеспечения для создания и тестирования алгоритмов шифрования важно соблюдать правила безопасной работы:

1. Проверить уровень освещенности рабочего места.
2. Проверить состояние электрооборудования.
3. Проверить исправность элементов электросистемы, обеспечивающей питание компьютера, включая электропроводку, выключатели, вилки и розетки, при помощи которых аппаратура подключается к сети.
4. Проконтролировать заземление компьютера.
5. Проверить его работоспособность.
6. При обнаружении неисправностей оборудования, сообщите руководителю.
